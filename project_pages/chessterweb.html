CHESSTER Website

Chroniciling CHESSTER. First Time Doing
Front-End

The index page. Designs, such as the background, were
made in Figma.

Slideshows on the Mechanical page displaying the CADs
for the board. Descriptions for each image are displayed
as a list under the slides.

Electrical page. The left side contains a chessboard, with
squares having a 'peekaboo effect'. Clicking shows a
step in the board's electrical flow.

Chesster was an AI-powered chessboard that instructs the player to move pieces by lighting up the board squares with LEDs. The project encompassed the elaborate marriage between an array of mechanical, electrical, and software components. We wanted to document the logic and process behind Chesster, and what better way to do that than a website?

Building this website was my first ever experience with front-end development. It was created using HTML, CSS, Javascript, and numerous tutorials; it was deployed with Github. If you want to see the website, click this link! The biggest challenge was implementing the more animated features of the website, such as the slideshows. These features bugged out quite a bit during implementation, and we had to mess around with the margins and padding to fix these bugs.
